Item: PAY-102

- claimed_risk: low (verified: medium)
- claimed_impact: medium
- area_hint: core (payments)

## Approach

Adds a `service_fee` field to `/payments/summary`, computed as a fixed
1.5% of `captured_total`. Per reviewer feedback, verified risk is
medium, which requires the new behavior to be gated behind a feature
flag (default off). The field is now only included in the response
when `payments_service_fee` (added to `flags.json`, default `false`)
is enabled via `app.flags.enabled(...)`. Tests cover both flag states
(field absent when off, present and correctly computed when on).
